Introduction


We aim to investigate cyber autonomy among government websites. Gathering top 100 for each Group of Seven (G7) countries, 3 crucial elements of security concerns are measured below:

HTTPS

Any en-route, off-path adversary are capable of hijacking route.

Certificate Authority

Rogue CAs can easily spoof certificates.

Content Provider

Loading resources from untrusted server or foreign server, chances are that malicious code will be executed.
